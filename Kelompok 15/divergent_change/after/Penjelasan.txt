Pada kode ini terdapat sebuah code smell yang disebut Divergent Change. Artinya, class Rectangle melakukan lebih dari satu pekerjaan atau memiliki tanggung jawab yang berbeda-beda. Selain melakukan operasi terhadap Rectangle seperti setter dan getter untuk width, height, dan lain-lain, class ini juga memiliki kemampuan untuk melakukan print(). Code smell ini tidak hanya disebabkan karena class Rectangle melakukan lebih dari satu pekerjaan, tetapi juga karena method print() tidak relevan dengan class Rectangle itu sendiri.

Untuk mengatasi masalah ini, dapat dilakukan Extract Class, yaitu dengan membuat class baru seperti BorderRectanglePrinter dan FullRectanglePrinter. Method print dari class Rectangle kemudian dipindahkan ke kedua class baru tersebut. Dengan begitu, tanggung jawab masing-masing class menjadi lebih spesifik dan terdefinisi dengan baik, sehingga memudahkan dalam pengembangan dan perawatan kode di masa depan.
