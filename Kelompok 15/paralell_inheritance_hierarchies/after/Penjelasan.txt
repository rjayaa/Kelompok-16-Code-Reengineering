Pada kode ini terdapat code smell yang disebut parallel inheritance hierarchies. Artinya, ketika kita membuat class shape baru seperti Rectangle, kita juga harus membuat class RectangleAreaCalculator yang mengimplementasi interface AreaInterface. Hal ini menyebabkan adanya hirarki paralel dalam kode, yang dapat mengakibatkan kompleksitas dan kesulitan dalam pengembangan dan perawatan kode di masa depan.

Untuk mengatasi masalah ini, dapat dilakukan refactoring dengan membuat class Shape2D mengimplementasi interface AreaInterface. Dengan begitu, setiap class yang meng-extend class Shape2D seperti Rectangle dan Triangle akan secara otomatis memiliki method area(). Dengan demikian, kita tidak perlu lagi membuat class khusus seperti RectangleAreaCalculator setiap kali membuat class shape baru. Hal ini memudahkan dalam pengembangan dan perawatan kode karena tidak perlu lagi memikirkan class kalkulator untuk setiap bentuk baru yang dibuat.
